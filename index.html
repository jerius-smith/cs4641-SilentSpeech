Silent Speech Detection

CS 4641 Spring 2020

Motivation

Silent Speech Detection aims to detect language when no sound is transmitted. This technology can be implemented in many use cases, such as private communication between soldiers on a battle field, or interacting with conversational AI, like Siri or Alexa, in noisy environments. Research has gone into using many different mechanisms to capture data from the mouth. Some have used surface electromyography and video. But we would like to use a Tongue Board, a retainer-like device which can be used to track the position of of tongue. Unlike other devices, the Tongue Board is partially invisible and can be used when walking.
